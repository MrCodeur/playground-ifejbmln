Observe o trecho do programa C abaixo.
``` 
 if (X >= Y){
   W=0;
 }  
else { 
  W=1;
}
O Valor armazenado na variável W será igual a 0 se:
() X >= Y
() X > Y
() X < Y
() X <= Y
() X = Y
() Nenhuma das anteriores
